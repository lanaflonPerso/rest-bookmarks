Bookmarks Web API

A Web API that allows you to store and share bookmarks online. A bookmark is a URL associated with a name and a description.

This service is only deployed for educational purposes. More specifically, this service demonstrates the usage of REST constraints, hypermedia and some HTTP advanced features like content negotiation, conditional requests and cache management.

All resources created with this API are only temporarily stored and the total amount of bookmarks is limited. Hence, once the limit is reached, a new resource will automatically destroy the oldest one.
